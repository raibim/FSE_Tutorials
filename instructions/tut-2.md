# Tutorial 2

In this tutorial, we will start building the core functionality of the _Wealth-Wise_ Dashboard application. We will focus on implementing functions to handle financial transactions, calculate balances, and manage budgets.

## Completing the Transactions Module
The `transactions.py` module contains several functions that need to be implemented. Follow the instructions below to complete these functions.

Each function has a `# TODO` comment indicating what needs to be done. Make sure to read the _docstrings_ for each function to understand its purpose and expected behavior.


## Completing the tutorial
Once you have completed the functions in the `transactions.py` module, you can test your implementation by running the command `pytest tests/test_transactions.py` in your terminal. This will run the unit tests provided for the transactions module and help you verify that your implementations are correct.

Make sure all tests pass successfully before moving on to the next tutorial. If any tests fail, review your code and the test cases to identify and fix the issues.


Happy coding!
